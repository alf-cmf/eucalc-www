The Global Calculator - Guidance for creating a pathway

Guidance for creating a pathway

  1. Set the levers. Click on the boxes to set each lever. Each lever has four boxes, as described below. Some users take an example pathway as a starting point and change the lever choices from there.
  2. Check cumulative emissions. If you want to create a pathway consistent with international commitments to limit global mean temperature increase to 2C, then keep cumulative emissions below the line for a 50% chance of achieving this.
  3. Check warnings. Potential problems with your pathway are highlighted in red in the warnings panel. Hover over the icons to view suggested solutions.
